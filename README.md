本项目基于模拟电商业务场景，独立完成了用户、商品、订单、支付、行为日志等多张业务表的设计，并使用 Python 生成原始数据集；随后将 CSV 数据导入 MySQL，完成数据落库与核心主题表构建；在此基础上，基于 SQL 搭建 ODS、DWD、DWS、ADS 数仓分层体系，开发订单明细层、用户行为明细层及用户、商品、订单日汇总表；进一步围绕新增用户数、DAU、GMV、支付转化率、商品销量/销售额 Top10、渠道下单分布、用户转化漏斗等核心指标开展分析，最终结合 pandas、matplotlib、Pillow 自动生成多张业务分析图表，并拼接输出电商用户行为分析总览看板，用于项目展示与结果汇报。
